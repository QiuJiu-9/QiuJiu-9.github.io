# z
<font size=4>这个属性决定了弹幕显示的层级，默认为'W'。从低到高为：</font>

|  系统层级  | 弹幕层级代码 |
| :--------: | :----------: |
|            |     'P'      |
|   远景图   |              |
|            |     'M'      |
|  地图图块  |              |
|            |     'E'      |
| 事件和玩家 |              |
|            |     'W'      |
| 天气和图片 |              |
|            |     'A'      |

<font size=4>完整的属性范例:   </font>

z:'P',   
z:'M',   
z:'E',   
z:'W',   
z:'A',
