# imgStretchMode
<font size=4>当激光需要变得长一些时，激光的图片如何变化，默认值是 'C'.</font>

| Command |            Note            | Example |
| :-----: | :------------------------: | :-----: |
|   'C'   | 激光的图片会直接缩放来变长 |   'C'   |
|   'T'   | 激光的图片会进行平铺来变长 |   'T'   |

<font size=4>完整的属性范例:    </font>

imgStretchMode:'C',   
imgStretchMode:'T',
