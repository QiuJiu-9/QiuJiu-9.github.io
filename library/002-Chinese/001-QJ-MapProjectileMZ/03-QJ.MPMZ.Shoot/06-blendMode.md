# blendMode
<font size=4>这个属性决定了弹幕图片的合成方式。    </font>

| Command |     Note      |
| :-----: | :-----------: |
|    0    |  NORMAL 正常  |
|    1    |   ADD 叠加    |
|    2    | MULTIPLY 相乘 |
|    3    |  SCREEN 画面  |

<font size=4>完整的属性范例:   </font>

blendMode:0,   
blendMode:1,   
blendMode:2,   
blendMode:3,
